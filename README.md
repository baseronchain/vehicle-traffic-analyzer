# Traffic Vehicle Classifier

This project is an intelligent traffic monitoring system designed to detect,
classify, and count vehicles in real-time using computer vision techniques.

The system classifies vehicles into four categories:
- Car
- Motorcycle
- Bus
- Truck

## Features
- Vehicle detection and tracking
- Multi-class vehicle classification
- Traffic flow counting
- Real-time video processing

## Applications
- Traffic monitoring and analysis
- Smart transportation systems
- Urban planning and research
